# The Cult - Modern E-Commerce Platform

## 🚀 Overview
A full-stack e-commerce platform built with modern technologies.

## 🛠️ Tech Stack
- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Node.js, Express, Prisma
- **Database**: PostgreSQL
- **Authentication**: JWT
- **Payments**: Stripe

## 📁 Project Structure

the-cult/

└── database/     # Database configuration
